Java жҠҖжңҜиғҪеӨҹдёҖзӣҙдҝқжҢҒйқһеёёеҘҪзҡ„еҗ‘еҗҺе…је®№жҖ§пјҢиҝҷзӮ№ Class ж–Үд»¶з»“жһ„зҡ„зЁіе®ҡжҖ§еҠҹдёҚеҸҜжІЎгҖӮJava е·Із»ҸеҸ‘еұ•еҲ° 14 зүҲжң¬пјҢдҪҶжҳҜ class ж–Үд»¶з»“жһ„зҡ„еҶ…е®№пјҢз»қеӨ§йғЁеҲҶеңЁJDK1.2 ж—¶д»Је°ұе·Із»Ҹе®ҡд№үеҘҪдәҶгҖӮиҷҪз„¶ JDK1.2 зҡ„еҶ…е®№жҜ”иҫғеҸӨиҖҒпјҢдҪҶжҳҜ java еҸ‘еұ•з»ҸеҺҶдәҶеҚҒдҪҷдёӘеӨ§зүҲжң¬пјҢдҪҶжҳҜжҜҸж¬Ўеҹәжң¬дёҠзҹҘиҜҶеңЁеҺҹжңүз»“жһ„еҹәзЎҖдёҠж–°еўһеҶ…е®№гҖҒжү©е……еҠҹиғҪпјҢе№¶жңӘеҜ№е®ҡд№үзҡ„еҶ…е®№еҒҡдҝ®ж”№гҖӮ

д»»дҪ•дёҖдёӘ Class ж–Үд»¶йғҪеҜ№еә”зқҖе”ҜдёҖдёҖдёӘзұ»жҲ–жҺҘеҸЈзҡ„е®ҡд№үдҝЎжҒҜпјҢдҪҶеҸҚиҝҮжқҘиҜҙпјҢClass ж–Үд»¶е®һйҷ…дёҠе®ғе№¶дёҚдёҖе®ҡд»ҘзЈҒзӣҳж–Үд»¶зҡ„еҪўејҸеӯҳеңЁпјҲжҜ”еҰӮеҸҜд»ҘеҠЁжҖҒз”ҹжҲҗгҖҒжҲ–иҖ…зӣҙжҺҘйҖҒе…Ҙзұ»еҠ иҪҪеҷЁдёӯпјүгҖӮ

Class ж–Үд»¶жҳҜдёҖз»„д»Ҙ 8 дҪҚеӯ—иҠӮдёәеҹәзЎҖеҚ•дҪҚзҡ„дәҢиҝӣеҲ¶жөҒгҖӮ

е·Ҙе…·д»Ӣз»Қ

SublimeпјҡжҹҘзңӢ 16 иҝӣеҲ¶зҡ„зј–иҫ‘еҷЁ
javapпјҡjavap жҳҜ JDK иҮӘеёҰзҡ„еҸҚи§Јжһҗе·Ҙе…·гҖӮе®ғзҡ„дҪңз”ЁжҳҜе°Ҷ .class еӯ—иҠӮз Ғж–Үд»¶и§ЈжһҗжҲҗеҸҜиҜ»зҡ„ж–Үд»¶ж јејҸгҖӮ
еңЁдҪҝз”Ё javap ж—¶жҲ‘дёҖиҲ¬дјҡж·»еҠ  -v еҸӮж•°пјҢе°ҪйҮҸеӨҡжү“еҚ°дёҖдәӣдҝЎжҒҜгҖӮеҗҢж—¶пјҢжҲ‘д№ҹдјҡдҪҝз”Ё -p еҸӮж•°пјҢжү“еҚ°дёҖдәӣз§Ғжңүзҡ„еӯ—ж®өе’Ңж–№жі•гҖӮ
jclasslibпјҡеҰӮжһңдҪ дёҚеӨӘд№ жғҜдҪҝз”Ёе‘Ҫд»ӨиЎҢзҡ„ж“ҚдҪңпјҢиҝҳеҸҜд»ҘдҪҝз”Ё jclasslibпјҢjclasslib жҳҜдёҖдёӘеӣҫеҪўеҢ–зҡ„е·Ҙе…·пјҢиғҪеӨҹжӣҙеҠ зӣҙи§Ӯзҡ„жҹҘзңӢеӯ—иҠӮз Ғдёӯзҡ„еҶ…е®№гҖӮе®ғиҝҳеҲҶй—ЁеҲ«зұ»зҡ„еҜ№зұ»дёӯзҡ„еҗ„дёӘйғЁеҲҶиҝӣиЎҢдәҶж•ҙзҗҶпјҢйқһеёёзҡ„дәәжҖ§еҢ–гҖӮеҗҢж—¶пјҢе®ғиҝҳжҸҗдҫӣдәҶ Idea зҡ„жҸ’д»¶пјҢдҪ еҸҜд»Ҙд»Һ plugins дёӯжҗңзҙўеҲ°е®ғгҖӮ

Class ж–Үд»¶ж јејҸ

д»ҺдёҖдёӘ Class ж–Үд»¶ејҖе§ӢпјҢж•ҙдёӘ Class ж–Үд»¶зҡ„ж јејҸе°ұжҳҜдёҖдёӘдәҢиҝӣеҲ¶зҡ„еӯ—иҠӮжөҒгҖӮеҗ„дёӘж•°жҚ®йЎ№зӣ®дёҘж јжҢүз…§йЎәеәҸзҙ§еҮ‘ең°жҺ’еҲ—еңЁ Class ж–Үд»¶д№ӢдёӯпјҢдёӯй—ҙжІЎжңүж·»еҠ д»»дҪ•еҲҶйҡ”з¬ҰпјҢиҝҷдҪҝеҫ—ж•ҙдёӘ Class ж–Үд»¶дёӯеӯҳеӮЁзҡ„еҶ…е®№еҮ д№Һе…ЁйғЁжҳҜзЁӢеәҸиҝҗиЎҢзҡ„еҝ…иҰҒж•°жҚ®пјҢжІЎжңүз©әйҡҷеӯҳеңЁгҖӮ

Class ж–Үд»¶ж јејҸйҮҮз”ЁдёҖз§Қзұ»дјјдәҺ C иҜӯиЁҖз»“жһ„дҪ“зҡ„дјӘз»“жһ„жқҘеӯҳеӮЁж•°жҚ®пјҢиҝҷз§ҚдјӘз»“жһ„дёӯеҸӘжңүдёӨз§Қж•°жҚ®зұ»еһӢпјҡж— з¬ҰеҸ·ж•°е’ҢиЎЁгҖӮ

ж— з¬ҰеҸ·ж•°еұһдәҺеҹәжң¬зҡ„ж•°жҚ®зұ»еһӢпјҢд»Ҙ u1гҖҒu2гҖҒu4гҖҒu8 жқҘеҲҶеҲ«д»ЈиЎЁ 1 дёӘеӯ—иҠӮпјҲдёҖдёӘеӯ—иҠӮжҳҜз”ұдёӨдҪҚ 16 иҝӣеҲ¶ж•°з»„жҲҗ пјҲ1дёӘ16иҝӣеҲ¶ж•°=4дёӘдәҢиҝӣеҲ¶ж•° 8дёӘдәҢиҝӣеҲ¶ж•°=дёҖдёӘеӯ—иҠӮпјүпјүгҖҒ2 дёӘеӯ—иҠӮгҖҒ4 дёӘеӯ—иҠӮе’Ң 8 дёӘеӯ—иҠӮзҡ„ж— з¬ҰеҸ·ж•°пјҢж— з¬ҰеҸ·ж•°еҸҜд»Ҙз”ЁжқҘжҸҸиҝ°ж•°еӯ—гҖҒзҙўеј•еј•з”ЁгҖҒж•°йҮҸеҖјжҲ–иҖ…жҢүз…§ UTF-8 зј–з Ғжһ„жҲҗеӯ—з¬ҰдёІеҖјгҖӮ

иЎЁжҳҜз”ұеӨҡдёӘж— з¬ҰеҸ·ж•°жҲ–иҖ…е…¶д»–иЎЁдҪңдёәж•°жҚ®йЎ№жһ„жҲҗзҡ„еӨҚеҗҲж•°жҚ®зұ»еһӢпјҢжүҖжңүиЎЁйғҪд№ жғҜжҖ§ең°д»ҘвҖң_infoвҖқз»“е°ҫгҖӮиЎЁз”ЁдәҺжҸҸиҝ°жңүеұӮж¬Ўе…ізі»зҡ„еӨҚеҗҲз»“жһ„зҡ„ж•°жҚ®пјҢж•ҙдёӘClass ж–Үд»¶жң¬иҙЁдёҠе°ұжҳҜдёҖеј иЎЁгҖӮ

Class ж–Үд»¶ж јејҸиҜҰи§Ј

Class зҡ„з»“жһ„дёҚеғҸ XML зӯүжҸҸиҝ°иҜӯиЁҖпјҢз”ұдәҺе®ғжІЎжңүд»»дҪ•еҲҶйҡ”з¬ҰеҸ·пјҢжүҖд»ҘеңЁе…¶дёӯзҡ„ж•°жҚ®йЎ№пјҢж— и®әжҳҜйЎәеәҸиҝҳжҳҜж•°йҮҸпјҢйғҪжҳҜиў«дёҘж јйҷҗе®ҡзҡ„пјҢе“ӘдёӘеӯ—иҠӮд»ЈиЎЁд»Җд№Ҳеҗ«д№үпјҢй•ҝеәҰжҳҜеӨҡе°‘пјҢе…ҲеҗҺйЎәеәҸеҰӮдҪ•пјҢйғҪдёҚе…Ғи®ёж”№еҸҳгҖӮ

жҢүйЎәеәҸеҢ…жӢ¬пјҡ

йӯ”ж•°дёҺ Class ж–Үд»¶зҡ„зүҲжң¬

жҜҸдёӘ Class ж–Үд»¶зҡ„еӨҙ 4 дёӘеӯ—иҠӮз§°дёәйӯ”ж•°пјҲMagic NumberпјүпјҢе®ғзҡ„е”ҜдёҖдҪңз”ЁжҳҜзЎ®е®ҡиҝҷдёӘж–Үд»¶жҳҜеҗҰдёәдёҖдёӘиғҪиў«иҷҡжӢҹжңәжҺҘеҸ—зҡ„ Class ж–Үд»¶гҖӮдҪҝз”Ёйӯ”ж•°иҖҢдёҚжҳҜжү©еұ•еҗҚжқҘиҝӣиЎҢиҜҶеҲ«дё»иҰҒжҳҜеҹәдәҺе®үе…Ёж–№йқўзҡ„иҖғиҷ‘пјҢеӣ дёәж–Үд»¶жү©еұ•еҗҚеҸҜд»ҘйҡҸж„Ҹең°ж”№еҠЁгҖӮж–Үд»¶ж јејҸзҡ„еҲ¶е®ҡиҖ…еҸҜд»ҘиҮӘз”ұең°йҖүжӢ©йӯ”ж•°еҖјпјҢеҸӘиҰҒиҝҷдёӘйӯ”ж•°еҖјиҝҳжІЎжңүиў«е№ҝжіӣйҮҮз”ЁиҝҮеҗҢж—¶еҸҲдёҚдјҡеј•иө·ж··ж·ҶеҚіеҸҜгҖӮ

зҙ§жҺҘзқҖйӯ”ж•°зҡ„ 4 дёӘеӯ—иҠӮеӯҳеӮЁзҡ„жҳҜ Class ж–Үд»¶ зҡ„зүҲжң¬еҸ·пјҡз¬¬ 5 е’Ңз¬¬ 6 дёӘеӯ—иҠӮжҳҜж¬ЎзүҲжң¬еҸ·пјҲMinorVersionпјүпјҢз¬¬ 7 е’Ңз¬¬ 8 дёӘеӯ—иҠӮжҳҜдё»зүҲжң¬еҸ·пјҲMajor VersionпјүгҖӮ

Java зҡ„зүҲжң¬еҸ·жҳҜд»Һ 45 ејҖе§Ӣзҡ„пјҢJDK 1.1 д№ӢеҗҺзҡ„жҜҸдёӘ JDK еӨ§зүҲжң¬еҸ‘еёғдё»зүҲжң¬еҸ·еҗ‘дёҠеҠ  1 й«ҳзүҲжң¬зҡ„ JDK иғҪеҗ‘дёӢе…је®№д»ҘеүҚзүҲжң¬зҡ„ Class ж–Үд»¶пјҢдҪҶдёҚиғҪиҝҗиЎҢд»ҘеҗҺзүҲжң¬зҡ„ Class ж–Үд»¶пјҢеҚідҪҝж–Үд»¶ж јејҸе№¶жңӘеҸ‘з”ҹд»»дҪ•еҸҳеҢ–пјҢиҷҡжӢҹжңәд№ҹеҝ…йЎ»жӢ’з»қжү§иЎҢи¶…иҝҮе…¶зүҲжң¬еҸ·зҡ„ Class ж–Үд»¶гҖӮд»ЈиЎЁ JDK1.8пјҲ16 иҝӣеҲ¶зҡ„ 34пјҢжҚўжҲҗ 10 иҝӣеҲ¶е°ұжҳҜ 52пјү

еёёйҮҸжұ 

еёёйҮҸжұ дёӯеёёйҮҸзҡ„ж•°йҮҸжҳҜдёҚеӣәе®ҡзҡ„пјҢжүҖд»ҘеңЁеёёйҮҸжұ зҡ„е…ҘеҸЈйңҖиҰҒж”ҫзҪ®дёҖйЎ№ u2 зұ»еһӢзҡ„ж•°жҚ®пјҢд»ЈиЎЁеёёйҮҸжұ е®№йҮҸи®Ўж•°еҖјпјҲconstant_pool_countпјүгҖӮдёҺ Java дёӯиҜӯиЁҖд№ жғҜдёҚдёҖж ·зҡ„жҳҜпјҢиҝҷдёӘе®№йҮҸи®Ўж•°жҳҜд»Һ 1 иҖҢдёҚжҳҜ 0 ејҖе§Ӣзҡ„

еёёйҮҸжұ дёӯдё»иҰҒеӯҳж”ҫдёӨеӨ§зұ»еёёйҮҸпјҡеӯ—йқўйҮҸпјҲLiteralпјүе’Ңз¬ҰеҸ·еј•з”ЁпјҲSymbolic ReferencesпјүгҖӮ
еӯ—йқўйҮҸжҜ”иҫғжҺҘиҝ‘дәҺ Java иҜӯиЁҖеұӮйқўзҡ„еёёйҮҸжҰӮеҝөпјҢеҰӮж–Үжң¬еӯ—з¬ҰдёІгҖҒеЈ°жҳҺдёә final зҡ„еёёйҮҸеҖјзӯүгҖӮ
з¬ҰеҸ·еј•з”ЁеҲҷеұһдәҺзј–иҜ‘еҺҹзҗҶж–№йқўзҡ„жҰӮеҝөпјҢеҢ…жӢ¬дәҶдёӢйқўдёүзұ»еёёйҮҸпјҡзұ»е’ҢжҺҘеҸЈзҡ„е…Ёйҷҗе®ҡеҗҚпјҲFully Qualified NameпјүгҖҒеӯ—ж®өзҡ„еҗҚз§°е’ҢжҸҸиҝ°з¬ҰпјҲDescriptorпјүгҖҒж–№жі•зҡ„еҗҚз§°е’ҢжҸҸиҝ°з¬Ұ

и®ҝй—®ж Үеҝ—

з”ЁдәҺиҜҶеҲ«дёҖдәӣзұ»жҲ–иҖ…жҺҘеҸЈеұӮж¬Ўзҡ„и®ҝй—®дҝЎжҒҜпјҢеҢ…жӢ¬пјҡиҝҷдёӘ Class жҳҜзұ»иҝҳжҳҜжҺҘеҸЈпјӣжҳҜеҗҰе®ҡд№үдёә public зұ»еһӢпјӣжҳҜеҗҰе®ҡд№үдёә abstract зұ»еһӢпјӣеҰӮжһңжҳҜзұ»зҡ„иҜқпјҢжҳҜеҗҰиў«еЈ°жҳҺдёә final зӯү

зұ»зҙўеј•гҖҒзҲ¶зұ»зҙўеј•дёҺжҺҘеҸЈзҙўеј•йӣҶеҗҲ

иҝҷдёүйЎ№ж•°жҚ®жқҘзЎ®е®ҡзұ»зҡ„з»§жүҝе…ізі»гҖӮзұ»зҙўеј•з”ЁдәҺзЎ®е®ҡиҝҷдёӘзұ»зҡ„е…Ёйҷҗе®ҡеҗҚпјҢзҲ¶зұ»зҙўеј•з”ЁдәҺзЎ®е®ҡиҝҷдёӘзұ»зҡ„зҲ¶зұ»зҡ„е…Ёйҷҗе®ҡеҗҚгҖӮз”ұдәҺ Java иҜӯиЁҖдёҚе…Ғи®ёеӨҡйҮҚз»§жүҝпјҢжүҖд»ҘзҲ¶зұ»зҙўеј•еҸӘжңүдёҖдёӘпјҢйҷӨдәҶ java.lang.Object д№ӢеӨ–пјҢжүҖжңүзҡ„ Java зұ»йғҪжңүзҲ¶зұ»пјҢеӣ жӯӨйҷӨдәҶjava.lang.Object еӨ–пјҢжүҖжңү Java зұ»зҡ„зҲ¶зұ»зҙўеј•йғҪдёҚдёә 0гҖӮжҺҘеҸЈзҙўеј•йӣҶеҗҲе°ұз”ЁжқҘжҸҸиҝ°иҝҷдёӘзұ»е®һзҺ°дәҶе“ӘдәӣжҺҘеҸЈпјҢиҝҷдәӣиў«е®һзҺ°зҡ„жҺҘеҸЈе°ҶжҢү implements иҜӯеҸҘпјҲеҰӮжһңиҝҷдёӘзұ»жң¬иә«жҳҜдёҖдёӘжҺҘеҸЈпјҢеҲҷеә”еҪ“жҳҜ extends иҜӯеҸҘпјүеҗҺзҡ„жҺҘеҸЈйЎәеәҸд»Һе·ҰеҲ°еҸіжҺ’еҲ—еңЁжҺҘеҸЈзҙўеј•йӣҶеҗҲдёӯ

жҲ‘д»¬е…Ҳд»ҘHashMapејҖе§Ӣеҗ§пјҢдҪ иғҪи®Іи®ІеҪ“дҪ newдёҖдёӘHashMapзҡ„ж—¶еҖҷпјҢдјҡеҸ‘з”ҹд»Җд№Ҳеҗ—пјҹ

  1. еҰӮжһңжҲ‘д»¬дёҚжҢҮе®ҡпјҢй»ҳи®ӨHashMapзҡ„еӨ§е°Ҹдёә16пјҢиҙҹиҪҪеӣ еӯҗзҡ„еӨ§е°Ҹдёә0.75
  2. иҝҳжңүе°ұжҳҜпјҡHashMapзҡ„еӨ§е°ҸеҸӘиғҪжҳҜ2ж¬Ўе№Ӯзҡ„пјҢеҒҮи®ҫдҪ дј дёҖдёӘ10иҝӣеҺ»пјҢе®һйҷ…дёҠжңҖз»ҲHashMapзҡ„еӨ§е°ҸжҳҜ16пјҢдҪ дј дёҖдёӘ7иҝӣеҺ»пјҢHashMapжңҖз»Ҳзҡ„еӨ§е°ҸжҳҜ8пјҢе…·дҪ“зҡ„е®һзҺ°еңЁtableSizeForеҸҜд»ҘзңӢеҲ°гҖӮ
  3. жҲ‘д»¬жҠҠе…ғзҙ ж”ҫиҝӣHashMapзҡ„ж—¶еҖҷпјҢйңҖиҰҒз®—еҮәиҝҷдёӘе…ғзҙ жүҖеңЁзҡ„дҪҚзҪ®пјҲhashпјү
  4. еңЁHashMapйҮҢз”Ёзҡ„жҳҜдҪҚиҝҗз®—жқҘд»ЈжӣҝеҸ–жЁЎпјҢиғҪеӨҹжӣҙеҠ й«ҳж•Ҳең°з®—еҮәиҜҘе…ғзҙ жүҖеңЁзҡ„дҪҚзҪ®
  5. дёәд»Җд№ҲHashMapзҡ„еӨ§е°ҸеҸӘиғҪжҳҜ2ж¬Ўе№ӮпјҢеӣ дёәеҸӘжңүеӨ§е°Ҹдёә2ж¬Ўе№Ӯж—¶пјҢжүҚиғҪеҗҲзҗҶз”ЁдҪҚиҝҗз®—жӣҝд»ЈеҸ–жЁЎгҖӮ
  6. иҖҢиҙҹиҪҪеӣ еӯҗзҡ„еӨ§е°ҸеҶіе®ҡзқҖе“ҲеёҢиЎЁзҡ„жү©е®№е’Ңе“ҲеёҢеҶІзӘҒгҖӮ
  7. жҜ”еҰӮзҺ°еңЁжҲ‘й»ҳи®Өзҡ„HashMapеӨ§е°Ҹдёә16пјҢиҙҹиҪҪеӣ еӯҗдёә0.75пјҢиҝҷж„Ҹе‘ізқҖж•°з»„жңҖеӨҡеҸӘиғҪж”ҫ12дёӘе…ғзҙ пјҢдёҖж—Ұи¶…иҝҮ12дёӘе…ғзҙ пјҢеҲҷе“ҲеёҢиЎЁйңҖиҰҒжү©е®№
  8. жҖҺд№Ҳз®—еҮәжҳҜ12е‘ўпјҹеҫҲз®ҖеҚ•пјҢе°ұжҳҜ16*0.7 5гҖӮжҜҸж¬Ўputе…ғзҙ иҝӣеҺ»зҡ„ж—¶еҖҷпјҢйғҪдјҡжЈҖжҹҘHashMapзҡ„еӨ§е°ҸжңүжІЎжңүи¶…иҝҮиҝҷдёӘйҳҲеҖјпјҢеҰӮжһңжңүпјҢеҲҷйңҖиҰҒжү©е®№гҖӮ
  9. йүҙдәҺдёҠйқўзҡ„иҜҙжі•пјҲHashMapзҡ„еӨ§е°ҸеҸӘиғҪжҳҜ2ж¬Ўе№ӮпјүпјҢжүҖд»Ҙжү©е®№зҡ„ж—¶еҖҷж—¶еҖҷй»ҳи®ӨжҳҜжү©еҺҹжқҘзҡ„2еҖҚ
  10. иҝҳжңүе°ұжҳҜжү©е®№иҝҷдёӘж“ҚдҪңиӮҜе®ҡжҳҜиҖ—ж—¶зҡ„пјҢйӮЈиғҪдёҚиғҪжҠҠиҙҹиҪҪеӣ еӯҗи°ғй«ҳдёҖзӮ№пјҢжҜ”еҰӮжҲ‘иҰҒи°ғиҮідёә1пјҢйӮЈжҲ‘зҡ„HashMapе°ұзӯүеҲ°16дёӘе…ғзҙ зҡ„ж—¶еҖҷжүҚжү©е®№е‘ўгҖӮ
  11. еҪ“з„¶жҳҜеҸҜд»Ҙзҡ„пјҢдҪҶжҳҜдёҚжҺЁиҚҗгҖӮиҙҹиҪҪеӣ еӯҗи°ғй«ҳдәҶпјҢиҝҷж„Ҹе‘ізқҖе“ҲеёҢеҶІзӘҒзҡ„жҰӮзҺҮдјҡеўһй«ҳпјҢе“ҲеёҢеҶІзӘҒжҰӮзҺҮеўһй«ҳпјҢеҗҢж ·дјҡиҖ—ж—¶пјҲеӣ дёәжҹҘжүҫзҡ„йҖҹеәҰеҸҳж…ўдәҶпјү

з®—дәҶгҖӮжҲ‘иҝҳжғіз»§з»ӯй—®дёӢпјҢеңЁputе…ғзҙ зҡ„ж—¶еҖҷпјҢдј йҖ’зҡ„KeyжҳҜжҖҺд№Ҳз®—е“ҲеёҢеҖјзҡ„пјҹ

  1. е®һзҺ°е°ұеңЁhashж–№жі•дёҠпјҢеҸҜд»ҘеҸ‘зҺ°зҡ„жҳҜпјҢе®ғжҳҜе…Ҳз®—еҮәжӯЈеёёзҡ„е“ҲеёҢеҖјпјҢз„¶еҗҺдёҺй«ҳ16дҪҚеҒҡејӮжҲ–иҝҗз®—пјҢдә§з”ҹжңҖз»Ҳзҡ„е“ҲеёҢеҖјгҖӮ
  2. иҝҷж ·еҒҡзҡ„еҘҪеӨ„еҸҜд»ҘеўһеҠ дәҶйҡҸжңәжҖ§пјҢеҮҸе°‘дәҶзў°ж’һеҶІзӘҒзҡ„еҸҜиғҪжҖ§гҖӮ

дҪ з®ҖеҚ•еҶҚиҜҙдёӢputе’Ңgetж–№жі•зҡ„е®һзҺ°еҗ§

  1. еңЁputзҡ„ж—¶еҖҷпјҢйҰ–е…ҲеҜ№keyеҒҡhashиҝҗз®—пјҢи®Ўз®—еҮәиҜҘkeyжүҖеңЁзҡ„indexгҖӮ
  2. еҰӮжһңжІЎзў°ж’һпјҢзӣҙжҺҘж”ҫеҲ°ж•°з»„дёӯпјҢеҰӮжһңзў°ж’һдәҶпјҢйңҖиҰҒеҲӨж–ӯзӣ®еүҚж•°жҚ®з»“жһ„жҳҜй“ҫиЎЁиҝҳжҳҜзәўй»‘ж ‘пјҢж №жҚ®дёҚеҗҢзҡ„жғ…еҶөжқҘиҝӣиЎҢжҸ’е…ҘгҖӮ
  3. еҒҮи®ҫkeyжҳҜзӣёеҗҢзҡ„пјҢеҲҷжӣҝжҚўеҲ°еҺҹжқҘзҡ„еҖјгҖӮжңҖеҗҺеҲӨж–ӯе“ҲеёҢиЎЁжҳҜеҗҰж»ЎдәҶпјҲеҪ“еүҚе“ҲеёҢиЎЁеӨ§е°Ҹ*иҙҹиҪҪеӣ еӯҗпјүпјҢеҰӮжһңж»ЎдәҶпјҢеҲҷжү©е®№
  4. еңЁgetзҡ„ж—¶еҖҷпјҢиҝҳжҳҜеҜ№keyеҒҡhashиҝҗз®—пјҢи®Ўз®—еҮәиҜҘkeyжүҖеңЁзҡ„indexпјҢз„¶еҗҺеҲӨж–ӯжҳҜеҗҰжңүhashеҶІзӘҒ
  5. еҒҮи®ҫжІЎжңүеҶІзӘҒзӣҙжҺҘиҝ”еӣһпјҢеҒҮи®ҫжңүеҶІзӘҒеҲҷеҲӨж–ӯеҪ“еүҚж•°жҚ®з»“жһ„жҳҜй“ҫиЎЁиҝҳжҳҜзәўй»‘ж ‘пјҢеҲҶеҲ«д»ҺдёҚеҗҢзҡ„ж•°жҚ®з»“жһ„дёӯеҸ–еҮәгҖӮ

йӮЈеңЁHashMapдёӯжҳҜжҖҺд№ҲеҲӨж–ӯдёҖдёӘе…ғзҙ жҳҜеҗҰзӣёеҗҢзҡ„е‘ўпјҹ

  1. йҰ–е…ҲдјҡжҜ”иҫғhashеҖјпјҢйҡҸеҗҺдјҡз”Ё==иҝҗз®—з¬Ұе’ҢequalsпјҲпјүжқҘеҲӨж–ӯиҜҘе…ғзҙ жҳҜеҗҰзӣёеҗҢгҖӮ
  2. иҜҙзҷҪдәҶе°ұжҳҜпјҡеҰӮжһңеҸӘжңүhashеҖјзӣёеҗҢпјҢйӮЈиҜҙжҳҺиҜҘе…ғзҙ е“ҲеёҢеҶІзӘҒдәҶпјҢеҰӮжһңhashеҖје’ҢequalsпјҲпјү|| ==йғҪзӣёеҗҢпјҢйӮЈиҜҙжҳҺиҜҘе…ғзҙ жҳҜеҗҢдёӘгҖӮ

дҪ иҜҙHashMapзҡ„ж•°жҚ®з»“жһ„жҳҜж•°з»„+й“ҫиЎЁ/зәўй»‘ж ‘пјҢйӮЈд»Җд№Ҳжғ…еҶөжӢҝдёӢжүҚдјҡз”ЁеҲ°зәўй»‘ж ‘е‘ўпјҹ

  1. еҪ“ж•°з»„зҡ„еӨ§е°ҸеӨ§дәҺ64дё”й“ҫиЎЁзҡ„еӨ§е°ҸеӨ§дәҺ8зҡ„ж—¶еҖҷжүҚдјҡе°Ҷй“ҫиЎЁж”№дёәзәўй»‘ж ‘пјҢеҪ“зәўй»‘ж ‘еӨ§е°Ҹдёә6ж—¶пјҢдјҡйҖҖеҢ–дёәй“ҫиЎЁгҖӮ
  2. иҝҷйҮҢиҪ¬зәўй»‘ж ‘йҖҖеҢ–дёәй“ҫиЎЁзҡ„ж“ҚдҪңдё»иҰҒеҮәдәҺжҹҘиҜўе’ҢжҸ’е…Ҙж—¶еҜ№жҖ§иғҪзҡ„иҖғйҮҸгҖӮ
  3. й“ҫиЎЁжҹҘиҜўж—¶й—ҙеӨҚжқӮеәҰOпјҲNпјүпјҢжҸ’е…Ҙж—¶й—ҙеӨҚжқӮеәҰOпјҲ1пјүпјҢзәўй»‘ж ‘жҹҘиҜўе’ҢжҸ’е…Ҙж—¶й—ҙеӨҚжқӮеәҰOпјҲlogNпјү

дҪ еңЁж—ҘеёёејҖе§ӢдёӯLinkedHashMapз”Ёзҡ„еӨҡеҗ—пјҹ

  1. еңЁеүҚйқўд№ҹжҸҗеҲ°дәҶпјҢLinkedHashMapеә•еұӮз»“жһ„жҳҜж•°з»„+й“ҫиЎЁ+еҸҢеҗ‘й“ҫиЎЁпјҢе®һйҷ…дёҠе®ғз»§жүҝдәҶHashMapпјҢеңЁHashMapзҡ„еҹәзЎҖдёҠз»ҙжҠӨдәҶдёҖдёӘеҸҢеҗ‘й“ҫиЎЁ
  2. жңүдәҶиҝҷдёӘеҸҢеҗ‘й“ҫиЎЁпјҢжҲ‘д»¬зҡ„жҸ’е…ҘеҸҜд»ҘжҳҜжңүеәҸзҡ„пјҢиҝҷйҮҢзҡ„жңүеәҸдёҚжҳҜжҢҮеӨ§е°ҸжңүеәҸиҖҢжҳҜжҸ’е…ҘжңүеәҸ
  3. LinkedHashMapеңЁйҒҚеҺҶзҡ„ж—¶еҖҷе®һйҷ…з”Ёзҡ„жҳҜеҸҢеҗ‘й“ҫиЎЁжқҘйҒҚеҺҶзҡ„пјҢжүҖд»ҘLinkedHashMapзҡ„еӨ§е°ҸдёҚдјҡеҪұе“ҚеҲ°йҒҚеҺҶзҡ„жҖ§иғҪ

йӮЈTreeMapе‘ўпјҹ

  1. TreeMapеңЁзҺ°е®һејҖеҸ‘дёӯз”Ёеҫ—д№ҹдёҚеӨҡпјҢTre eMapзҡ„еә•еұӮж•°жҚ®з»“жһ„жҳҜзәўй»‘ж ‘
  2. TreeMapзҡ„keyдёҚиғҪдёәnullпјҲеҰӮжһңдёәnullпјҢйӮЈиҝҳжҖҺд№ҲжҺ’еәҸе‘ўпјүпјҢTreeMapжңүеәҸжҳҜйҖҡиҝҮComparatorжқҘиҝӣиЎҢжҜ”иҫғзҡ„пјҢеҰӮжһңcomparatorдёәnullпјҢйӮЈд№Ҳе°ұдҪҝз”ЁиҮӘз„¶йЎәеәҸ

еҶҚжқҘи®Іи®ІзәҝзЁӢе®үе…Ёзҡ„Mapеҗ§пјҹ HashMapжҳҜзәҝзЁӢе®үе…Ёзҡ„еҗ—пјҹ

  1. HashMapдёҚжҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢеңЁеӨҡзәҝзЁӢзҺҜеўғдёӢпјҢHashMapжңүеҸҜиғҪдјҡжңүж•°жҚ®дёўеӨұе’ҢиҺ·еҸ–дёҚдәҶжңҖж–°ж•°жҚ®зҡ„й—®йўҳпјҢжҜ”еҰӮиҜҙпјҡзәҝзЁӢAputиҝӣеҺ»дәҶпјҢзәҝзЁӢBgetдёҚеҮәжқҘгҖӮ
  2. жғіиҰҒзәҝзЁӢе®үе…ЁпјҢдёҖиҲ¬дҪҝз”ЁConcurrentHashMap
  3. ConcurrentHashMapжҳҜзәҝзЁӢе®үе…Ёзҡ„Map е®һзҺ°зұ»пјҢе®ғеңЁjucеҢ…дёӢзҡ„гҖӮ
  4. зәҝзЁӢе®үе…Ёзҡ„Mapе®һзҺ°зұ»йҷӨдәҶConcurrentHashMapиҝҳжңүдёҖдёӘеҸ«еҒҡHashtableгҖӮ
  5. еҪ“з„¶дәҶпјҢд№ҹеҸҜд»ҘдҪҝз”ЁCollectionsжқҘеҢ…иЈ…еҮәдёҖдёӘзәҝзЁӢе®үе…Ёзҡ„MapгҖӮ
  6. дҪҶж— и®әжҳҜHashtableиҝҳжҳҜCollectionsеҢ…иЈ…еҮәжқҘзҡ„йғҪжҜ”иҫғдҪҺж•ҲпјҲеӣ дёәжҳҜзӣҙжҺҘеңЁеӨ–еұӮеҘ—synchronizeпјүпјҢжүҖд»ҘжҲ‘д»¬дёҖиҲ¬жңүзәҝзЁӢе®үе…Ёй—®йўҳиҖғйҮҸзҡ„пјҢйғҪдҪҝз”ЁConcurrentHashMap
  7. ConcurrentHashMapзҡ„еә•еұӮж•°жҚ®з»“жһ„жҳҜж•°з»„+й“ҫиЎЁ/зәўй»‘ж ‘пјҢе®ғиғҪж”ҜжҢҒй«ҳе№¶еҸ‘зҡ„и®ҝй—®е’Ңжӣҙж–°пјҢжҳҜзәҝзЁӢе®үе…Ёзҡ„гҖӮ
  8. ConcurrentHashMapйҖҡиҝҮеңЁйғЁеҲҶеҠ й”Ғе’ҢеҲ©з”ЁCASз®—жі•жқҘе®һзҺ°еҗҢжӯҘпјҢеңЁgetзҡ„ж—¶еҖҷжІЎжңүеҠ й”ҒпјҢNodeйғҪз”ЁдәҶvolatileз»ҷдҝ®йҘ°гҖӮ
  9. еңЁжү©е®№ж—¶пјҢдјҡз»ҷжҜҸдёӘзәҝзЁӢеҲҶй…ҚеҜ№еә”зҡ„еҢәй—ҙпјҢе№¶дё”дёәдәҶйҳІжӯўputValеҜјиҮҙж•°жҚ®дёҚдёҖиҮҙпјҢдјҡз»ҷзәҝзЁӢзҡ„жүҖиҙҹиҙЈзҡ„еҢәй—ҙеҠ й”Ғ

  1. Spring AOPи§ЈеҶізҡ„жҳҜйқһдёҡеҠЎд»Јз ҒжҠҪеҸ–зҡ„й—®йўҳ
  2. AOPеә•еұӮзҡ„жҠҖжңҜжҳҜеҠЁжҖҒд»ЈзҗҶпјҢеңЁSpring еҶ…е®һзҺ°дҫқиө–зҡ„жҳҜBeanPostProcessor
  3. жҜ”еҰӮжҲ‘д»¬йңҖиҰҒеңЁж–№жі•дёҠжіЁе…ҘдәӣгҖҢйҮҚеӨҚжҖ§гҖҚзҡ„йқһдёҡеҠЎд»Јз ҒпјҢе°ұеҸҜд»ҘеҲ©з”ЁSpring AOP
  4. жүҖи°“зҡ„гҖҢйқўеҗ‘еҲҮйқўзј–зЁӢгҖҚеңЁжҲ‘зҗҶи§ЈдёӢе…¶е®һе°ұжҳҜеңЁж–№жі•еүҚеҗҺеўһеҠ йқһдёҡеҠЎд»Јз Ғ

  2. дёҖдёӘзі»з»ҹзҰ»дёҚејҖзӣ‘жҺ§пјҢзӣ‘жҺ§еҹәжң¬зҡ„жҢҮж ҮжңүQPSгҖҒRTгҖҒERRORзӯүзӯү

  4. дәҺжҳҜжҲ‘еҲ©з”ЁжіЁи§Ј+AOPзҡ„ж–№ејҸе°ҒиЈ…дәҶдёҖжҠҠпјҢеҸӘиҰҒж–№жі•/зұ»дёҠеёҰжңүжҲ‘иҮӘе®ҡд№үзҡ„жіЁи§Ј
  5. ж–№жі•иў«и°ғз”Ёж—¶пјҢе°ұдјҡдёҠжҠҘAQSгҖҒRTзӯүдҝЎжҒҜе®һзҺ°дәҶйқһдёҡеҠЎд»Јз ҒдёҺдёҡеҠЎд»Јз ҒеҲҶзҰ»зҡ„ж•Ҳжһң

дәҶи§ЈпјҢдҪ д»¬йЎ№зӣ®дёҖиҲ¬жҳҜжҖҺд№ҲжҠҠеҜ№иұЎдәӨз»ҷIOCе®№еҷЁз®ЎзҗҶзҡ„пјҹ

  1. SpringжҸҗдҫӣдәҶ4з§Қж–№ејҸпјҢеҲҶеҲ«жҳҜпјҡ
    1пјүпјҡжіЁи§Ј2пјүпјҡXML3пјүпјҡJavaConfig 4пјүпјҡеҹәдәҺGroovy зҡ„DSLй…ҚзҪ®
  2. дёҖиҲ¬йЎ№зӣ®жҲ‘д»¬з”ЁжіЁи§ЈжҲ–XMLжҜ”иҫғеӨҡпјҢе°‘йғЁеҲҶз”ЁJavaConfig
  3. ж—ҘеёёеҶҷдёҡеҠЎд»Јз ҒдёҖиҲ¬з”ЁжіЁи§ЈжқҘе®ҡд№үеҗ„з§ҚеҜ№иұЎпјҢиҙЈд»»й“ҫиҝҷз§ҚдёҖиҲ¬й…ҚзҪ®еңЁXMLгҖҢжіЁи§ЈгҖҚи§ЈеҶідёҚдәҶзҡ„е°ұз”ЁJavaConfig
  4. жҖ»дҪ“иҖҢиЁҖпјҢиҝҳжҳҜеҫ—зңӢйЎ№зӣ®зҡ„д»Јз ҒйЈҺж јеҗ§
  5. еҸҚжӯЈе°ұжҳҜе®ҡд№үе…ғж•°жҚ®пјҢиғҪз»ҷеҲ°Springи§Јжһҗе°ұеҘҪдәҶ
